This is the current branch of this project.

UCSD Arthur C. Clarke Center 2013

Shortcuts:

Shift - Hold on Startup to display resolution dialog
N - Rename the server in the master server list
L - Toggle Leaderboard
I - Display IP Address
Q - Show QR Code
End - Single Player Mode
J - Add Jellyfish
+ - Go to Next Level
- - Go to Previous Level
F1 - Toggle Cursor Visibility


Camera

C - Toggle Gallery Camera
< - Zoom in Camera (same as scroll wheel)
> - Zoom out camera (same as scroll wheel)
